# LanguageToolThis is a repo for managing machine readable artifacts for the [LanguageTool APIs](http://www.languagetool.org/http-api/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [LanguageTool APIs](http://www.languagetool.org/http-api/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://www.languagetool.org/http-api/)- [Developer](http://www.languagetool.org/http-api/)- [Twitter](https://twitter.com/languagetoolorg)The [APIs.json](https://github.com/api-evangelist/languagetool/blob/master/apis.json) index provides links to all the relevant links for the LanguageTool APIs, including the OpenAPI and Postman if they are present.
